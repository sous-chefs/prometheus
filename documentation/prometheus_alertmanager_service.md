# prometheus_alertmanager_service

Manages the Alertmanager systemd unit.

## Actions

* `:create`
* `:enable`
* `:start`
* `:restart`
* `:reload`
* `:stop`
* `:delete`

## Properties

Common properties include `binary`, `config_file`, `storage_path`, `external_url`, `log_level`, `user`, and `group`.
