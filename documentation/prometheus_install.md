# prometheus_install

Installs Prometheus from an upstream binary archive, shell-managed archive extraction, or source checkout.

## Actions

* `:install`

## Properties

Common properties include `install_dir`, `log_dir`, `user`, `group`, `use_existing_user`, `install_method`, `version`, `binary_url`, `checksum`, `file_extension`, `source_repository`, `source_revision`, `config_file`, and `storage_path`.

`install_method` accepts `binary`, `shell_binary`, or `source`.
